# Java Alura — Exercícios e Desafios

Repositório com exercícios e desafios práticos dos cursos de **Java** da [Alura](https://www.alura.com.br).

## Conteúdo

Coleção de projetos independentes organizados por tema:

| Pasta | Tema |
|-------|------|
| `animal/` | Herança e polimorfismo |
| `carros/` | Classes e objetos |
| `contasBancarias/` | Contas correntes e bancárias |
| `ConversorDeMoeda/` | APIs de câmbio |
| `desafioBuscador/` | Consulta de CEP via API |
| `ExerciciosComApi/` | Integração com APIs externas |
| `NumerosPrimos/` | Algoritmos e lógica |
| `PersitenciaDeDados/` | Leitura e escrita de arquivos |
| `Tabuada/` | Estruturas de repetição |
| E outros... | POO, records, interfaces |

## Tecnologias

- Java (versões 11–21 conforme o exercício)
- Gson, HTTP Client, arquivos JSON/TXT

## Como usar

Cada pasta em `src/exercicios/` é um projeto independente. Abra a classe `Principal` (ou equivalente) no seu IDE e execute.

```bash
git clone https://github.com/Agsterr/Java-Alura.git
cd Java-Alura
```

Não há build unificado — compile e rode cada exercício separadamente.
